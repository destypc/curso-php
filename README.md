## 📖 Sobre o Projeto

Este repositório foi criado para armazenar todos os exercícios, desafios e projetos desenvolvidos durante meus estudos de **PHP**.

O objetivo é registrar minha evolução na linguagem, praticar conceitos importantes e construir uma base sólida para o desenvolvimento web back-end.

---

## 🎯 Objetivos

- Aprender a sintaxe do PHP
- Trabalhar com variáveis e tipos de dados
- Utilizar estruturas condicionais
- Utilizar estruturas de repetição
- Criar funções
- Manipular arrays
- Trabalhar com formulários
- Integrar PHP com HTML
- Preparar o caminho para estudos de banco de dados
